1.通过Centos安装zabbix-client端

我们的环境是Centos 7.4,服务器:192.168.76.129,客户端:192.168.76.128

1.1 安装依赖环境

[root@server share]# yum install gcc-c++ -y 
\\这里我用的默认yum源

1.2 下载zabbix软件包并解压

[root@client share]# wget http://sourceforge.net/projects/zabbix/files/ZABBIX%20Latest%20Stable/3.2.6/zabbix-3.2.6.tar.gz 
[root@client share]# tar -xvf zabbix-3.2.6.tar.gz

1.3 编译zabbix

[root@client share]# cd zabbix-3.2.6
[root@client zabbix-3.2.6]# ./configure --prefix=/usr/local/zabbix --enable-agent
[root@client zabbix-3.2.6]# make -j4 && make -j4 install
\\没有报错就安装成功了

2.修改配置文件并启动zabbix客户端

2.1 修改配置文件选项

[root@client zabbix-3.2.6]# sed -i 's/127.0.0.1/192.168.134.128/g;s/Zabbix server/192.168.134.129/g' /usr/local/zabbix/etc/zabbix_agentd.conf

2.2 拷贝启动脚本

[root@client zabbix-3.2.6]# cd misc/init.d/tru64/
[root@client tru64]# cp * /etc/init.d/
[root@client tru64]# ln -s /usr/local/zabbix/sbin/zabbix_agentd /usr/local/sbin/
[root@client tru64]# chmod +x /etc/init.d/zabbix_*
[root@client tru64]# cd -

2.3 启动zabbix客户端

[root@client zabbix-3.2.6]# useradd -s /sbin/nologin zabbix
[root@client zabbix-3.2.6]# /etc/init.d/zabbix_agentd start
Zabbix agent started.
